## 🎯 ACCÈS RAPIDE AUX SECTIONS

### 1️⃣ DASHBOARD (Vous êtes ici)

**Que faire ?**
- Voir vos statistiques
- Cliquer sur **"Start Creating"** pour créer
- Utiliser les **Quick Actions** pour naviguer

---

### 2️⃣ CRÉER UNE GÉNÉRATION

**3 façons d'y accéder** :
1. Cliquer sur **"Start Creating"** dans le Dashboard
2. Cliquer sur **"New Generation"** dans la sidebar (bouton bleu foncé)
3. Cliquer sur le bouton **➕ Plus** en haut

**Flow** :
```
Type Select → Intent Input → Analyzing → 
Direction Select → Analysis → CocoBoard → Generation → Result
```

---

### 3️⃣ VOIR L'HISTORIQUE 📜

**Comment** :
- Cliquer sur **"History"** dans la sidebar (icône 🕐)
- OU cliquer sur **"View Analytics"** dans Dashboard

**Actions** :
- 🔍 Rechercher
- 🏷️ Filtrer (All/Image/Video/Campaign)
- 📥 Download
- 🗑️ Delete

---

### 4️⃣ GÉRER L'ÉQUIPE 👥 (Enterprise)

**Comment** :
- Cliquer sur **"Team"** dans la sidebar
- OU cliquer sur **"Invite Team Members"** dans Dashboard

**Badge rouge** = nombre d'approbations en attente

---

### 5️⃣ GÉRER LES CRÉDITS ⚡

**Comment** :
- Cliquer sur **"Credits"** dans la sidebar

**Actions** :
- Voir le solde
- Acheter des packs
- Voir l'historique

---

### 6️⃣ PARAMÈTRES ⚙️

**Comment** :
- Cliquer sur **"Settings"** dans la sidebar

**Options** :
- Notifications
- Sons
- Auto-save

---
